# Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow, including data loading, cleaning, exploratory data analysis (EDA), SQL querying, and dashboard creation. The goal is to extract meaningful insights from raw data and present them in a clear, business-friendly format.

---

##  Dataset

* The dataset contains customer-related information such as purchases, categories, and transaction details.
* It is used to analyze trends, customer behavior, and key performance metrics.

---

##  Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **SQL** (PostgreSQL / MySQL / SQL Server)
* **Power BI** (Dashboard & Visualization)
* **Jupyter Notebook** (Data processing & analysis)
* **Gamma / PowerPoint** (Presentation)

---

## 🔄 Project Steps

### 1. Data Loading

* Imported dataset into Python using Pandas
* Inspected structure and data types

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized column names (converted to lowercase)
* Fixed inconsistent data formats

### 3. Exploratory Data Analysis (EDA)

* Analyzed distributions and trends
* Identified patterns and anomalies
* Visualized data using charts and graphs

### 4. SQL Analysis

* Loaded cleaned data into database
* Performed queries using:

  * Joins
  * Aggregations
  * Subqueries
  * Window Functions
* Extracted insights such as top products, customer segments, and trends

### 5. Dashboard Creation

* Connected database to Power BI
* Built interactive dashboard with:

  * KPIs
  * Filters (Slicers)
  * Charts (Bar, Line, Pie)

### 6. Reporting

* Created summary report with key insights
* Designed presentation (PPT) using Gamma

---

##  Dashboard

The dashboard provides:

* Sales trends over time
* Top-performing products
* Customer segmentation
* Discount analysis

---

## 📈 Results & Insights

* Identified top-selling products within categories
* Analyzed customer purchase behavior
* Measured discount impact on sales
* Provided actionable business insights

##  How to Run

1. Clone the repository
2. Install required Python libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn sqlalchemy
   ```
3. Run Jupyter Notebook for data cleaning & EDA
4. Load processed data into SQL database
5. Connect Power BI to database
6. Open Power BI file to view dashboard
